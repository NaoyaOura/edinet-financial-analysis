# テキスト分析対象セクションの選定根拠

## 対象セクション

有価証券報告書（第2部 第2 事業の状況）の以下3セクションを対象とする。

| # | セクション名 | 選定理由 |
|---|------------|---------|
| 1 | 経営方針、経営環境及び対処すべき課題等 | 経営トップがAI/DXをどう認識・位置づけているかが直接読み取れる |
| 2 | サステナビリティに関する考え方及び取組 | 近年AI/DX取組の開示が集中するセクション（2023年度から記載が義務化） |
| 4 | 経営者による財政状態、経営成績及びキャッシュ・フローの状況の分析（MD&A） | AI/DX投資が実際の経営成績にどう影響したかの記述が含まれる |

## 対象外とした主なセクション

| セクション | 除外理由 |
|-----------|---------|
| 第1部 企業情報（事業概要・沿革等） | 定型的な記述が多く、AI/DX意識を反映しにくい |
| 第5 経理の状況（財務諸表・注記） | 数値・会計基準の説明が中心で、テキストキーワード分析に適さない |
| 第3 設備の状況 | 設備投資の記述が中心で対象キーワードの出現頻度が低い |
| 第4 提出会社の状況（株式・役員等） | 企業のAI/DX活用意識とは無関係な記述が大半 |

## 備考

- 上記の選定は「企業のAI/DX活用意識・実態を拾いやすいか」を主な基準としている
- サステナビリティセクション（第2 事業の状況 > 2）は2023年3月期から記載が義務化されたため、それ以前の年度では存在しない書類がある点に注意
