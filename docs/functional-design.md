# 機能設計書

## 1. 研究目的

有価証券報告書における生成AI・DX関連キーワードの出現頻度と、日本企業の財務パフォーマンスとの関係を定量的に分析する。
メインの分析対象は小売業（「生成AIの活用が小売業の成長に寄与するか」の検証）とし、比較対象としてIT企業（情報通信業）も分析する。
IT企業は生成AI・DXとの親和性が高く、小売業との比較を通じて「キーワードスコアと業績の関係が業種によって異なるか」を検証する。

---

## 2. 研究仮説

| 仮説 | 内容 |
|---|---|
| H1（主仮説） | 生成AI・DX関連キーワードスコアが高い企業は、翌年の売上高成長率・営業利益率が高い |
| H2 | キーワードスコアの増加率が高い企業は、業績改善が見られる |
| H3（対立仮説） | キーワードスコアと業績に有意な関係はない（過剰なバズワード） |
| H4（業種比較） | キーワードスコアと業績の関係はIT企業において小売業よりも強く現れる |

---

## 3. 分析対象

### 3-1. 対象企業

- **国内企業のみ**（海外企業は対象外）
- **業種A（メイン）**: 日本標準産業分類に基づく小売業
  - 各種商品小売業（コンビニ・百貨店・スーパー等）
  - 織物・衣服・身の回り品小売業
  - 飲食料品小売業
  - 機械器具小売業（家電・車等）
  - その他の小売業
- **業種B（比較対象）**: 情報通信業（IT企業）
  - ソフトウェア業、情報処理サービス業、インターネット付随サービス業 等
  - **目的**: 生成AI・DXキーワードスコアが小売業の業績と有意な関係を示さない場合でも、IT企業の業績には強く影響するかを業種横断で比較検証するため
- **上場区分**: 金融商品取引法に基づきEDINETへ有価証券報告書を提出している企業
- **除外条件**: 分析期間中に上場廃止・業種変更した企業

### 3-2. 分析期間

| 区分 | 期間 | 目的 |
|---|---|---|
| ベースライン期間 | 2019〜2022年度 | 生成AI普及前の基準値 |
| 生成AI普及期 | 2023〜2024年度 | ChatGPT普及後の変化 |

> **注意（新型コロナウィルスの影響）**
> ベースライン期間（2019〜2022年度）は新型コロナウィルスの感染拡大と重なる。
> 特に小売業は外出自粛・消費行動の変化により業績への影響が大きいため、
> この期間を含む分析では以下の考慮が必要：
> - 2020〜2021年度をコロナ禍ダミー変数として回帰モデルに組み込む
> - コロナ前（2019年度）・コロナ禍（2020〜2021年度）・回復期（2022年度〜）でサブグループ分析を行う
> - IT企業は小売業ほど直接的な影響を受けていないため、業種間比較時にこの非対称性を考慮する

有価証券報告書の以下のセクションからテキストを抽出：

- **第2 事業の状況 > 1. 経営方針、経営環境及び対処すべき課題等**
- **第2 事業の状況 > 2. サステナビリティに関する考え方及び取組**
- **第2 事業の状況 > 4. 経営者による財政状態、経営成績及びキャッシュ・フローの状況の分析（MD&A）**

## 5. キーワード定義

### 5-1. キーワードカテゴリ

| カテゴリ | キーワード例 | 重み |
|---|---|---|
| 生成AI（強） | 生成AI、生成人工知能、ChatGPT、GPT-4、LLM、大規模言語モデル | 3 |
| AI全般（中） | AI、人工知能、機械学習、ディープラーニング、自然言語処理 | 2 |
| DX（弱） | DX、デジタルトランスフォーメーション、デジタル変革、デジタル化 | 1 |

### 5-2. キーワードスコア算出

```
キーワード密度 = Σ(出現回数 × 重み) ÷ 総文字数 × 10,000
```

- 総文字数で正規化することで、報告書の長さによるバイアスを排除
- 否定文脈（「導入していない」「検討していない」等）の前後N文字は除外

### 5-3. 企業グルーピング

各年度のキーワード密度を四分位で分類：

- **高スコア群（Q4）**: 上位25%
- **中スコア群（Q2-Q3）**: 中間50%
- **低スコア群（Q1）**: 下位25%

---

## 6. 分析手法

### 6-1. 基本分析：グループ比較

高・中・低スコア群の財務指標平均値を比較し、有意差を検定（t検定・ANOVA）。

### 6-2. ラグ回帰分析（メイン分析）

翌年の業績変化を目的変数、当年のキーワードスコアを説明変数とした回帰分析。

```
業績変化(t+1) = β₀ + β₁×キーワード密度(t) + β₂×log(売上高)(t) + β₃×業種ダミー + ε
```

- 1年ラグ・2年ラグの両方で実施
- コントロール変数：企業規模（売上高対数）、業種分類

### 6-3. 差分の差分法（DiD）

2023年を「処理期間」の開始として、キーワードが急増した企業群と増えなかった企業群を比較。

```
処理群：2023年度にキーワードスコアが前年比50%以上増加した企業
対照群：同期間にキーワードスコアの変化が軽微な企業
```

### 6-4. パネルデータ分析

固定効果モデルで企業固有の不変要因を除去した上で分析。

## 8. 制約・前提条件

| 項目 | 内容 |
|---|---|
| API利用 | EDINETのAPIキーが必要（事前取得）。キーは環境変数 `EDINET_API_KEY` から取得し、ソースコードへの直接記述・コミットは禁止 |
| データ形式 | XBRLタクソノミはEDINET標準タクソノミに準拠 |
| 言語 | 日本語テキストのみ対象（英語表記のAIは別途対応検討） |
| 除外業種 | 卸売業・製造業は対象外（小売業・IT企業のみ） |
| 新型コロナウィルスの影響 | 2020〜2021年度は小売業の業績に大きな外部ショックが発生しているため、この期間を含む分析ではコロナ禍ダミー変数の導入やサブグループ分析による感度確認を行う |
| 統計ライブラリ | Javaの標準ライブラリ範囲内 または Apache Commons Math |
