# ブレインストーミングメモ（2026-03-19）

## テーマ
生成AI・DXキーワードと日本小売企業の財務パフォーマンスの関係分析

## 中心的な問い
「有価証券報告書で生成AI・DX関連語を多用する日本の小売企業は、そうでない企業と比べて財務パフォーマンスが優れているか？」

## 因果の方向性
- 仮説A: DX投資 → 業績向上（因果）
- 仮説B: 業績が良い企業 → 余裕があるのでDX投資（逆因果）
- 仮説C: 大企業バイアス → 大企業は両方とも高い（交絡）
→ ラグ分析（前年のキーワード数 → 翌年または四半期の業績変化）で区別を試みる

## データ設計アイデア

### 取得対象
- EDINET登録の小売業（日本標準産業分類：大分類I〜J）
    - 小売業ではなくIT企業は業績に大きく寄与しているかも合わせて分析する
- 期間：2022〜2026年度
    - 2019年から2023年までは新型コロナウィルスの影響をうけていたため、その点も考慮にいれる
- 書類種別：有価証券報告書（docTypeCode=120）

### 財務指標候補
- 成長性：売上高成長率、営業利益成長率
- 収益性：売上高営業利益率、ROE、ROA
- 効率性：販管費率、人件費率
- 投資：設備投資額、無形固定資産

### キーワード設計
- 生成AI関連：生成AI、ChatGPT、GPT、LLM、大規模言語モデル 等
- DX関連：DX、デジタルトランスフォーメーション 等
- AI全般：AI、人工知能、機械学習 等
- 否定文脈は除外検討

## 分析手法候補
- グループ比較（高・中・低スコア群）
- ラグ回帰分析
- 差分の差分法（DiD）
- パネルデータ分析

## 研究の限界
- キーワード多い ≠ 実際に活用
- 大企業バイアス
- 業種内ばらつき
- タイムラグ
- 文書長さの差異
